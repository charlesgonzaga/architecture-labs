# Architecture Labs

Laboratório de Arquitetura de Software contendo estudos, exemplos e implementações práticas de padrões arquiteturais modernos.

Este repositório tem como objetivo demonstrar conhecimento em:

- Arquitetura de Software
- Sistemas Distribuídos
- Microsserviços
- Mensageria
- DDD
- CQRS
- Arquitetura Orientada a Eventos
